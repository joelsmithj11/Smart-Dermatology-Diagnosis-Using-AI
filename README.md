# Smart-Dermatology-Diagnosis-Using-AI

An AI-powered skin disease diagnosis system developed using Deep Learning, Computer Vision, Flask, and PyTorch. The system analyzes skin images and predicts possible skin diseases while providing a user-friendly interface for diagnosis and report generation.

## Project Overview

This project aims to assist in the early detection of skin diseases using artificial intelligence. Users can upload skin images and receive predictions generated by trained deep learning models.

## Features
- User Registration and Login System
- Role-Based Authentication (Admin & User)
- Skin Disease Detection from Images
- Deep Learning-Based Prediction
- Diagnosis History Tracking
- PDF Report Generation
- Analytics Dashboard
- Multilingual Support
- CAPTCHA Verification
- Grad-CAM Visualization for Explainable AI
- Responsive User Interface

## Dataset

The model was trained using a curated skin disease image dataset. Due to similarities among certain disease categories, some classes were merged to improve classification performance, resulting in 19 final disease classes.

## Installation
Clone the repository

git clone https://github.com/joelsmithj11/Smart-Dermatology-Diagnosis-Using-AI.git

Install dependencies

pip install -r requirements.txt

Download model files from the Google Drive link and place them inside the model folder.
Run the application

python run.py
